Users:
	id: unique meteor-generated user id
	createdAt: time created in ISO format
	services: Linked services (incl. password + resume login tokens)
	Service details: ex: facebook - all facebook information including access token, 		name, profile, location, etc.
	username: username

	Stuff to add for later:
		friends: (default null)
		following: (default null)
		followers: (default null)
		collection aka saved posts: (default null)

Posts (aka Thoughts):
	id: unique id
	text: post text
	createdAt - ISO date
	username: creator
	
	owner: owner id
	position: physical location of user (default null)
	randomIndex: random number (not necessarily unique) between 1 and 100000000

	collectedBy: {
       		type: [String],
       		label: "collectedBy",
       		max: 1000
   	}

	For Facebook posts:
		Also add facebook url, image/video url (if applicable), likes, type (status, wall post, etc.)

	Stuff to add:
		likes: (default null)
		rank: popularity rank, default 0
